## 🏗️ O Projeto: Visão de Engenharia e Negócio

O **Estudio Arae** é uma plataforma web concebida para apoiar e dar visibilidade ao trabalho independente de artistas e narradores gráficos [1]. Este projeto foi desenvolvido com foco na **experiência do usuário (UX)** e na **performance**, servindo como uma vitrine digital responsiva e fluida [1, 2].

Diferente de projetos puramente acadêmicos, o Estudio Arae resolve uma dor real: a necessidade de um espaço profissional e otimizado para a exposição de artes independentes [1, 3].

---

## 🛠️ Stack Tecnológica & Decisões Técnicas

Para este projeto, apliquei o conceito de **First Principles**, priorizando o domínio total sobre o ciclo de vida da aplicação antes da abstração para frameworks [4, 5].

*   **Lógica de Interatividade (JavaScript Vanilla):** Implementação de manipulação de DOM e eventos para criar uma experiência dinâmica sem o overhead de bibliotecas externas [2, 6, 7].
*   **Estilização e Layout (HTML5 & CSS3):** Arquitetura CSS focada em **Mobile-First** e **Design Responsivo**, garantindo acessibilidade em diversos dispositivos [2, 8, 9].
*   **Versionamento e Deploy (Git & GitHub Pages):** Utilização de Git para controle de versão e integração contínua para publicação automática [10, 11].
